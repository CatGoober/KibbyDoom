# Widescreen _WS Lump Support

Nyan Doom supports new `*_WS` background and statusbar lumps.

When any of the `widescreen` lumps are loaded within a WAD, they will take priority over the following `original` lumps.

### Doom Patches

| Original       | Widescreen              |
|----------------|-------------------------|
| STBAR          | STBAR_WS                |
| TITLEPIC       | TITLE_WS                |
| INTERPIC       | INTER_WS                |
| CREDIT         | CREDI_WS                |
| HELP           | HELP_WS                 |
| HELP1          | HELP1_WS                |
| HELP2          | HELP2_WS                |
| BOSSBACK       | BOSSB_WS                |
| WIMAP0         | WIMAP0WS                |
| WIMAP1         | WIMAP1WS                |
| WIMAP2         | WIMAP2WS                |
| VICTORY2       | VICTO_WS                |
| PFUB1          | PFUB1_WS                |
| PFUB2          | PFUB2_WS                |
| ENDPIC         | ENDPI_WS                |
